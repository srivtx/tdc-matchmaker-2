V2 of the TDC matchmaker workspace — same matching engine, same AI, same data, redesigned as an editorial morning brief. Mobile-first, custom icon family, dark and light modes. Full write-up in [WRITEUP.md](./WRITEUP.md).

```bash
npm install
cp .env.example .env.local
npm run dev
```

Login with `priya.sharma` / `tdc2024`.

---

**Stack** — Next.js 14 · TypeScript 5 · Tailwind 3 with CSS-variable design tokens · Instrument Serif · Inter · JetBrains Mono · Framer Motion · custom 1.25px-stroke SVG glyphs (no lucide, no material).
